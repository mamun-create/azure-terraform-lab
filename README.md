# Azure Terraform CI/CD Project

## 📌 Overview
This project demonstrates automated Azure infrastructure deployment using Terraform and GitHub Actions.  
It focuses on building a repeatable, consistent, and scalable Infrastructure as Code (IaC) workflow.

## ⚙️ CI/CD Pipeline
- GitHub Actions pipeline  
- Terraform init, validate, plan, apply  
- Automated deployment to Azure  

---

## 📊 Key Outcomes
- Reduced deployment time from ~30–40 mins to ~5–10 mins  
- Repeatable and consistent deployments  
- Reduced manual configuration errors  
- Faster environment rebuild and testing
